# ThreatLens AI

A cybersecurity threat monitoring and analytics platform built using Java, Spring Boot, React, and MySQL.

## Features

* Real-time Threat Dashboard
* Threat Severity Analytics
* Threat Status Tracking
* Threat Type Visualization
* Analyst Notes Management
* Threat Resolution Workflow
* CSV Report Export
* Login Authentication
* REST API Architecture

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* MySQL

### Frontend

* React
* Vite
* Axios
* Recharts

## Dashboard Features

* Threat Monitoring
* Brute Force Detection
* SQL Injection Tracking
* Malware Detection
* Severity Analytics
* Status Analytics

## Future Enhancements
